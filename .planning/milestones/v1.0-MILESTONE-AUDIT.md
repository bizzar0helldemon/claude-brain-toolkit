## Gap Analysis

All gaps share a single root cause: **setup.sh (Phase 3) was written before Phase 4 artifacts existed.** Phase 4 added two new files and one new hook type that setup.sh does not know about:

1. `hooks/post-tool-use.sh` — needs to be added to the deploy list
2. PostToolUse hook — needs to be added to the BRAIN_HOOKS merge string
3. `commands/brain-add-pattern.md` — needs deployment path (new directory for setup.sh)

The /brain-scan reference in brain-mode.md is a separate issue — it points to a skill that was never created in this milestone. It should either be implemented or the reference removed.

**Fix scope:** Update setup.sh only. No code changes needed to the hooks or skills themselves.
